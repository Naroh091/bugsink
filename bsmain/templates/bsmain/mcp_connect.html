Connect Claude Code

Connect Claude Code (or any MCP-compatible client) to this Bugsink instance so an LLM can autonomously browse issues, inspect stacktraces, and investigate errors.

You need an API token to authenticate. Replace <your-token> in the commands below with a token from the Tokens page.

Option 1 — one-time command

Run this once in your terminal. It writes the server into your local Claude Code config and applies only to your machine.

{% code %}:::bash claude mcp add --transport http bugsink {{ mcp_url }} \ --header "Authorization: Bearer " {% endcode %}

Option 2 — shared config (team)

This .mcp.json is already committed at the root of this repo. Each person sets BUGSINK_TOKEN in their shell and the server is picked up automatically.

{% code %}:::bash export BUGSINK_TOKEN= {% endcode %}

Add the line above to your ~/.bashrc or ~/.zshrc to make it permanent.

Verify the connection

Type /mcp in Claude Code — you should see bugsink · connected with 16 tools available.
